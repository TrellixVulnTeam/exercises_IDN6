4- Iconos

Los iconos se utilizan mediante una fuente de iconos creada con Icomoon.

Para utilizarlos disponemos de dos métodos:

  • En el HTML mediante las clases CSS especificadas en la siguiente tabla
  • En el SCSS mediante el mixin icon que recibe como parámetro el nombre del icono. Este mixin puede recibir más parámetros, puedes echar un ojo a su definición en css/base/_mixins.scss.

Siempre que se pueda se utilizarán clases. Se recurrirá al mixin cuando no quede más remedio
